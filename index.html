"진실을 추구함에 있어 우리는 편향되지 않아야 하지만, 진실을 발견했을 때 우리는 결코 중립적일 수 없다."

— 전설적인 저널리스트 월터 크론카이트(Walter Cronkite)의 신념을 잇는 뉴스 편향성 이원 분석기

가짜 뉴스 뿐만이 아니라 편향된 뉴스로 현재 대한민국은 언론의 도마 위에 있습니다. 언론의 편중과 확증 편향은 독자들로 하여금 하나의 좁은 버블(Filter Bubble) 속에 갇히게 만들고 있습니다.

이러한 현실에 입각하여 독자들에게 보다 객관적이고 균형 잡힌 뉴스를 전달하고, 미디어에 내재된 숨겨진 성향을 직관적으로 드러냄으로써 문제를 해결하고자 본 프로젝트를 만들었습니다.

저희 서비스는 주요 언론사의 뉴스를 통해 최대한 검증되지 않은 가짜 뉴스는 배제하고, 나아가 하나의 사건에 대한 관점을 여럿으로 보여줍니다.

각각의 뉴스 기사에 대해서는 사전 학습을 통해 정교하게 분리한 어휘 및 표현 기준을 잡아서, 해당 기사가 어떤 정치적 성향(진보·중도·보수)에 가까운지를 분석하고 비율(%)로 표시합니다.

뉴스를 클릭하면 해당 뉴스에 대한 한쪽으로 쏠리지 않은 편향되지 않은 시선으로 기사를 다각도에서 읽어볼 수 있도록 보조합니다.

해당 프로젝트는 단순히 한국어 뉴스를 분석하는 수준을 넘어서, 향후 해외 각국(특히 각 국가의 자국 시선)의 관점으로도 뉴스를 바라볼 수 있도록 다국적 시점을 제공하고자 합니다.

저희의 독자들이 편향의 버블을 깨고, 향후 더 넓게 이 세계라는 거대한 그림 속에서 다양한 길을 발견하기를 간절히 바랍니다.

미국에서 가장 신뢰받는 앵커였던 월터 크론카이트(Walter Cronkite). 취재 과정에서의 객관성과 진실 앞에서의 당당함을 상징한다.

Technical Composition & System Architecture

1. 기술 명세 및 시스템 스택

  • UI Framework: FastAPI (Uvicorn) 백엔드와 HTML5/CSS3 (Glassmorphism 다크 테마) 및 Vanilla JS 기반의 반응형 프론트엔드로 구현.
  • ML Classifier: 26개 주요 언론사 기사 데이터셋을 활용해 어휘(TF-IDF N-grams), 표현 특징, LDA 토픽 모델링을 반영하여 Logistic Regression 분류기로 학습. 모델 파일 크기 245KB (`bias_model.pkl`).
  • Generative AI: `google-genai` 라이브러리를 활용해 Gemini 2.5 Flash API 연동. 기사 편향 포인트 분석, 타 성향 언론사 관점 비교, 팩트체크 리포트 생성 및 실시간 패널 챗봇 질의응답 구현.
  • Data Source: 네이버 뉴스 OpenAPI를 통해 실시간 트렌딩 뉴스를 카테고리별로 자동 수집 및 추출.

2. 인프라 배포 및 완료 조건

  • 완료 조건: 사용자가 뉴스 기사를 선택하면 10초 이내에 기사의 진보/중도/보수 비율(%)이 그래프로 나타나고, Gemini가 생성한 4개 분석 리포트가 화면에 렌더링되며, 챗봇을 통해 실시간 질의응답이 가능하다.
  • 인프라 및 배포 트랙: AWS EC2 (Ubuntu 22.04 LTS) 인스턴스에 탄력적 IP를 매핑하고, Docker Compose를 사용하여 FastAPI 컨테이너와 Nginx 웹 프록시 서버를 통합 가동.
  • CI/CD 파이프라인: GitHub Actions 워크플로우를 활용해 코드 push 시 SSH 원격 명령으로 서버 무중단 배포 구현.
  • 12주차 n8n 자동화: 동일한 AWS EC2 서버 환경 하에 Docker Compose 및 Caddy를 묶어 n8n 워크플로우 자동화 엔진 병행 구축.
