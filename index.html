가짜 뉴스를 넘어 편향을 분석하다

가짜 뉴스 뿐만이 아니라 편향된 뉴스로 현재 대한민국은 언론의 도마 위에 있습니다. 이러한 현실에 입각하여 본 프로젝트는 독자들이 특정 진영의 생각에 갇히는 필터 버블(Filter Bubble) 현상을 깨뜨리고자 기획되었습니다.

저희 '뉴스밸런스 (NewsBalance)'는 주요 신문사를 통해 수집된 뉴스를 대상으로 작동하며, 무조건적인 가짜 뉴스 배제와 더불어 하나의 시사 이슈에 대해 다각적인 관점을 비교하여 볼 수 있는 기회를 선사합니다.

특히 본 프로젝트는 글로벌 뉴스 미디어 분석 플랫폼인 Ground News의 뉴스 성향 비교 방식에 착안하여 설계되었습니다. 나아가 Ground News의 단순 정보 분류 및 성향 배치를 넘어, 개별 기사 본문에 대한 실시간 한국어 머신러닝 편향 지수(%) 분류 파이프라인을 구축하고, 최신 생성 AI(Gemini)를 통한 팩트체크 리포트 및 에이전트 대화 챗봇을 융합함으로써 독자가 능동적으로 기사의 맥락을 짚어갈 수 있도록 종합 미디어 리터러시 시스템으로 고도화 및 업그레이드한 점이 특징입니다.

모든 수집 기사에 대해서는 사전 학습을 통하여 분리한 어휘 및 표현적 기준 데이터를 토대로 머신러닝 분류 알고리즘이 동작하며, 해당 기사가 진보·중도·보수 중 어느 성향에 가까운지를 분석하여 직관적인 백분율(%)로 표기합니다.

원하는 뉴스를 클릭하면 단순 수치 분석을 넘어, 편향성 완화를 위해 해당 뉴스를 균형 잡힌 시각에서 바라볼 수 있도록 보조하는 AI(Gemini) 종합 리포트 및 질의응답 챗봇 시스템을 함께 가동합니다.

향후 해당 프로젝트는 한국을 넘어서 향후 해외 각국, 특히 각 국가의 자국, 시선으로도 뉴스를 바라볼 수 있도록 시점을 제공하고자 합니다. 저희의 독자들이 향후 더 넓게 이 세계라는 거대한 그림 속에서 다양한 길을 발견하기를 바랍니다.

Technical Composition & System Architecture

1. [AI Core 1] 자체 머신러닝 (ML) 기반 편향 분류 파이프라인

  • 2만 건의 정치/사회 데이터셋 학습: 26개 주요 언론사의 실제 뉴스 본문을 수집하여 정밀 학습 말뭉치(Corpus)를 구축했습니다.
  • 복합 텍스트 특징 추출: 조사와 공백을 필터링한 뒤, 단어 흐름을 파악하는 TF-IDF 벡터화와 기사 전체의 내재된 주제를 추출하는 LDA 토픽 모델링을 결합해 다차원 확률 벡터를 생성합니다.
  • 경량화된 실시간 추론: 다중 클래스 로지스틱 회귀(Logistic Regression) 분류기를 통해 텍스트의 미세한 뉘앙스 차이를 포착하고 진보·중도·보수 성향을 즉각적으로 백분율화합니다. 모델을 245KB 크기로 압축 직렬화하여 클라우드 서버에서 지연 없는 추론이 가능합니다.

2. [AI Core 2] 생성형 AI(LLM) 다각도 논평 및 팩트체크 엔진

  • Gemini 2.5 Flash 실시간 연동: 단순 성향 분류를 넘어, 수집된 기사 본문을 프롬프트 엔지니어링을 통해 LLM 에이전트에 주입합니다.
  • 입체적 미디어 리터러시 리포트: 기사 내 잠재된 편향성 경고, 한쪽으로 치우치지 않기 위한 균형 잡힌 시각 제안, 타 성향 언론사들과의 프레이밍 비교, 객관적 팩트체크를 종합 리포트로 즉석에서 생성합니다.
  • 대화형 에이전트 챗봇: 독자가 기사를 읽다가 궁금한 점이나 의심스러운 프레임에 대해 챗봇에게 질문하면, LLM이 기사 팩트 기반으로 논리적이고 정중하게 의문을 해소해 줍니다.

3. [AI Core 3] GraphRAG 융합: 프레임 확산 그래프 시각화

  • 단편적 분석의 한계 극복: 하나의 거대한 정치적 사건(법안 발의, 스캔들)이 시간 흐름에 따라 언론사별로 어떻게 프레임이 변질되고 파생되는지를 추적합니다.
  • Entity-Relation 지식 그래프 구축: LLM을 통해 뉴스 텍스트에서 주체(Entity)와 행동(Relation)을 정밀 추출한 뒤, 이를 Neo4j Graph DB에 구조화된 데이터로 적재합니다.
  • 계층적 확산 경로 시각화: 2026년 대세 기술인 GraphRAG (Graph Retrieval-Augmented Generation) 아키텍처를 도입했습니다. Vis.js를 활용하여 노드와 엣지를 계층적(Hierarchical)으로 배치함으로써, A 언론사가 퍼트린 최초의 맥락이 타 매체로 퍼져나가며 오염되는 경로를 인터랙티브 차트로 증명합니다.

📡 클라우드 인프라 및 자동화 배포

  • 안전한 백엔드 아키텍처: FastAPI 비동기 서버를 기반으로 하며, 사용자 요청 대기 시간을 최소화하기 위해 백그라운드 스레드(BackgroundTasks)에서 지식 그래프 추출을 병행 처리합니다.
  • Docker 및 Nginx 역방향 프록시: AWS EC2 (Ubuntu) 환경에서 컨테이너 기반으로 격리 구동되며, 보안 헤더 및 정적 캐싱이 적용된 Nginx를 통해 외부 트래픽을 처리합니다.
  • CI/CD 파이프라인: GitHub Actions를 통한 무중단 자동화 배포가 연동되어 있어, 코드 푸시 즉시 서비스가 안정적으로 업데이트됩니다.
