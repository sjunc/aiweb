편향 분석 기준

뉴스밸런스(NewsBalance)는 26개 주요 언론사의 기사를 수집하여 해당 기사들이 실제 사용하는 어휘, 표현, 단어, 주제 등을 머신러닝으로 학습한 분류 모델을 기반으로, 임의의 뉴스 기사 하나하나에 대해 진보·중도·보수 성향 비율을 수치화합니다.

중요: 아래 분류표는 참고용으로, 각 언론사가 주로 사용하는 언어적 패턴의 경향성을 보여줍니다. 이는 특정 언론사에 대한 낙인이 아니라, 해당 기준으로 개별 기사의 어휘·표현·주제 등을 분석한 결과입니다. ML 분석 %는 언론사 성향과 별개로 기사 자체의 언어를 기준으로 산출됩니다.
