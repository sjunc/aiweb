# 뉴스밸런스 (NewsBalance)

> 11주차 — 기말 프로젝트 한 장 요약.

---

## 한 줄 요약

뉴스 기사 URL을 입력하거나 실시간 트렌딩 뉴스를 선택하면, 머신러닝 모델이 기사의 정치적 편향도를 수치화(%)하고 Gemini 2.5 Flash가 다각적 관점 및 팩트체크 리포트를 제공하는 미디어 리터러시 지원 웹 서비스.

## 문제

가짜 뉴스뿐만 아니라 특정 정치적 입장에 치우친 편향적인 뉴스의 범람으로 인해 언론에 대한 불신과 확증 편향(필터 버블)이 사회적 문제로 대두되고 있습니다. 독자들은 하나의 기사만 읽고 사실을 오인하거나 치우친 시각을 갖기 쉬우며, 객관적이고 다각적인 시각에서 사건을 바라볼 수 있는 보조 도구가 절대적으로 부족한 상황입니다.

---

## 사용자

뉴스 기사를 소비하면서 정치적 편향에서 벗어나 균형 잡힌 시각을 유지하고자 하는 일반 대중 및 미디어 리터러시에 관심이 높은 독자층.

---

## 핵심 기능 (3개 이하)

1. **실시간 트렌딩 뉴스 수집**: 네이버 뉴스 API를 통해 카테고리별(시사, 정치, 세계 등) 실시간 뉴스를 실시간으로 자동 수집하여 대시보드에 제공합니다.
2. **정치 편향성 ML 이원 분석**: 26개 주요 언론사의 대량의 어휘, 표현, 주제를 사전에 학습한 머신러닝 분류기(Logistic Regression + TF-IDF + LDA 토픽 모델링)를 통해 기사별 진보·중도·보수 성향 비율(%)을 수치화하여 시각적으로 시각화합니다.
3. **AI 기반 다각적 리포트 및 질의응답**: Gemini 2.5 Flash API를 활용하여 기사 본문의 언어적 편향 포인트를 지적하고, 타 성향 언론사의 보도 관점을 비교하며, 추가 팩트체크를 제공하고, AI 패널과 실시간으로 대화하며 질문할 수 있는 챗봇 기능을 제공합니다.

---

## 입력 → AI 처리 → 출력

사용자의 실시간 뉴스 클릭 또는 기사 내용 전달 → 로컬 머신러닝 분류기(TF-IDF + LDA + Logistic Regression)를 통해 정치 성향 3단계 비율(%) 도출 → Gemini 2.5 Flash API를 활용한 편향 포인트, 언론사 비교, 팩트체크 등의 심층 논평 생성 → 대시보드 화면에 시각화 차트 및 보고서 출력 및 챗봇 질의응답 창 활성화.

---

## 기술 스택

- **UI**: FastAPI (Uvicorn) + HTML5 / CSS3 (Glassmorphism 다크 테마) / Vanilla JS
- **모델 / API**: Gemini 2.5 Flash API (`google-genai`), scikit-learn (LogisticRegression, TfidfVectorizer, LatentDirichletAllocation)
- **외부 서비스**: 네이버 뉴스 검색 OpenAPI

---

## 데이터 출처

- **네이버 뉴스 OpenAPI 검색 API**: 실시간 트렌딩 뉴스 수집에 활용.
- **26개 주요 언론사별 기사 데이터셋**: `ml_classifier.py` 학습을 위해 네이버 검색 API를 통해 각 언론사별 기사 어휘 및 표현 패턴 데이터를 직접 크롤링하여 학습 데이터셋 구축.

---

## 완료 조건

사용자가 뉴스 리스트에서 기사를 클릭하면, **10초 이내**에 기사의 진보/중도/보수 비율(%)이 그래프로 나타나고, Gemini가 생성한 4개 항목(편향 포인트, 균형 시각, 언론사 비교, 팩트체크)의 분석 보고서가 지연 없이 화면에 렌더링되며, 챗봇을 통해 기사에 관한 질문을 던졌을 때 실시간으로 패널 답변이 출력되어야 한다.

- 피어 피드백 2: 다양한 성향의 언론사 사설 비교 기능이 추가되면 더 유익할 것 같습니다.
- 강사 판정: ○
- 강사 코멘트: 복잡한 형태소 분석기 없이 공백 기반 토크나이저 fallback과 가벼운 ML 파이프라인으로 훌륭히 성능을 구현했으며, Gemini API 연계를 통한 팩트체크와 챗봇 인터페이스가 유기적입니다.
